# The Shop API

> Web API designed for online shopping.

## Project Summary:
[Click me to see the summary](SUMMARY.md)

## Tool Chain:
+ Base Framework: Flask
+ ORM: SqlAlchemy
+ Serializer: Marshmallow
+ Server: gunicorn
+ Load-Balancer: nginx
+ DB: Mysql
+ UnitTest: pytest

## Project Setup:
Run the whole project 
```
make run
```

Run tests
```
make test
```
